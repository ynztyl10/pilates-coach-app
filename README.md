# 普拉提教练手册

一个基于 React + TypeScript + Tailwind CSS 的普拉提教学知识索引系统，面向普拉提教练和康复从业者。

## 功能

- **体态问题索引**：涵盖头颈肩、脊柱、骨盆、下肢等常见体态问题的运动解剖分析
- **训练方案**：每个条目提供系统的普拉提训练动作、要点和临床笔记
- **禁忌提示**：标注各问题的训练禁忌与注意事项
- **AI 助手**：内置智能问答界面（当前为模拟回复，可接入真实 LLM）
- **分类检索**：支持按部位分类筛选和关键词搜索

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- @iconify/tailwind

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 数据来源

知识库内容基于普拉提运动解剖学、临床康复经验和专业教材整理，仅供教学参考。
